电脑入门 02：文件怎么管才不乱

发表于 2026-03-21 更新于 2026-04-10 分类于 电脑入门 阅读次数：

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

文件和文件夹

文件就是一个个具体的东西：照片、文档、安装包。
文件夹就是装文件的“抽屉”。

一定要把后缀名打开

Windows默认隐藏后缀名，这是个大坑。

打开方法：随便开个文件夹 → 点“查看” → 勾选“文件扩展名”

这样你就能看到每个文件的真实身份了。
很多病毒会伪装成“照片.jpg.exe”，如果你只能看到“照片.jpg”，一点就中招。

文件放哪了？看路径

比如你存了一张照片在“D盘 → 我的文件 → 照片”里，它的位置就是：

这个叫路径，就是文件的“家庭住址”。

复制/剪切/粘贴

  • Ctrl + C 复制（原地留一份）
  • Ctrl + X 剪切（搬走）
  • Ctrl + V 粘贴（放到当前位置）

移动文件：剪切 → 到目标位置 → 粘贴
复制文件：复制 → 到目标位置 → 粘贴

桌面不是用来存文件的！

桌面放太多文件会：

  • 开机变慢
  • 重装系统时全丢

原则：桌面上只放快捷方式（左下角有小箭头那种），真正的文件都放到D盘或文档里。
