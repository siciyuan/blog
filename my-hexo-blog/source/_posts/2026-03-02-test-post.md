# 测试文章

欢迎来到我的Hexo博客！这是一篇测试文章，用于展示博客的基本功能。

## 什么是Hexo？

Hexo是一个快速、简洁且高效的静态站点生成框架。它基于Node.js，支持Markdown语法，可以帮助你快速创建个人博客、文档网站等。

## 为什么选择Hexo？

- **速度快**：Hexo生成静态文件的速度非常快，即使是大型网站也能在几秒内生成。
- **简洁**：Hexo的配置简单明了，容易上手。
- **强大**：支持丰富的插件和主题，可以根据需要定制。
- **Markdown支持**：使用Markdown编写文章，语法简洁易读。

## 如何创建新文章？

在命令行中执行以下命令：

```bash
hexo new "文章标题"
```

这会在`source/_posts`目录中创建一个新的Markdown文件，你可以在其中编写文章内容。

## 部署到GitHub

生成静态文件后，可以部署到GitHub Pages：

```bash
hexo deploy
```

这样，你的博客就可以通过GitHub Pages在线访问了。

## 总结

Hexo是一个非常适合个人博客的静态站点生成器，它简单易用，功能强大。希望这篇测试文章能帮助你了解Hexo的基本使用方法。

祝你使用Hexo愉快！
