## 多主题支持设置

### 方法1：手动切换主题

1. **下载其他主题**：
   - 访问 https://hexo.io/themes/ 浏览可用主题
   - 下载喜欢的主题，解压到 `themes/` 目录

2. **修改配置**：
   - 编辑 `_config.yml` 文件
   - 将 `theme: butterfly` 改为 `theme: 主题名称`

3. **重新生成**：
   - 运行 `hexo clean && hexo generate`
   - 运行 `hexo server` 查看效果

### 方法2：使用主题切换插件

1. **安装插件**：
   ```bash
   npm install hexo-theme-switcher --save
   ```

2. **配置插件**：
   在 `_config.yml` 中添加：
   ```yaml
   theme_switcher:
     theme_config:
       butterfly:
         name: Butterfly
         description: 现代响应式主题
       next:
         name: Next
         description: 简洁优雅主题
   ```

3. **添加切换按钮**：
   在主题布局文件中添加切换按钮
