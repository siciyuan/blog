常用软件安装教程

发表于 2026-04-14 更新于 2026-04-15 分类于 电脑 ， 教程

常用软件安装教程

本教程将指导您如何在 Windows 系统上安装一些常用的软件，包括 Java 开发环境、游戏平台 Steam、网络加速工具 Watt Toolkit 和 FastGithub，以及压缩软件 7-Zip 和歌声合成软件 OpenUtau。

1. Java (JDK) 安装

Java 是广泛使用的编程语言和计算平台。

安装步骤：

  1. 访问 Oracle 官网下载页面： https://www.oracle.com/java/technologies/downloads/
  2. 根据您的操作系统选择合适的 JDK 版本（通常选择最新 LTS 版本）
  3. 下载完成后，双击安装包开始安装
  4. 按照安装向导提示完成安装过程
  5. 配置环境变量（可选但推荐）：
    • 右键”此电脑” → “属性” → “高级系统设置”
    • 点击”环境变量”
    • 在系统变量中找到”Path”，点击”编辑”
    • 添加 JDK 的 bin 目录路径（例如：C:\Program Files\Java\jdk-17\bin）

2. Steam 安装

Steam 是全球最大的综合性数字发行平台之一。

安装步骤：

  1. 访问 Steam 官网： https://store.steampowered.com/
    (也可以用s.team)2. 点击页面右上角的”安装 Steam”按钮
  2. 下载 Steam 安装程序（SteamSetup.exe）
  3. 双击下载的安装文件运行安装向导
  4. 按照提示完成安装过程
  5. 安装完成后启动 Steam 并登录您的账户

3. Watt Toolkit (原名 Steam++) 安装

Watt Toolkit 是一款开源跨平台的多功能游戏工具箱，主要用于优化 Steam 等游戏平台的网络连接。

安装步骤：

  1. 访问项目 GitHub 页面： https://github.com/BeyondDimension/SteamTools
  2. 点击”Releases”标签查看最新版本
  3. 根据您的操作系统下载对应的安装包：
    • Windows: 下载 .exe 安装文件
    • macOS: 下载 .dmg 文件
    • Linux: 下载 .AppImage 或 .deb 文件
  4. 双击下载的安装包开始安装
  5. 按照安装向导完成安装过程
  6. 启动软件并根据需要进行配置

4. FastGithub 安装

FastGithub 是一个 GitHub 加速工具，可以解决 GitHub 访问慢的问题。

安装步骤：

  1. 访问项目 GitHub 页面： https://github.com/dotnetcore/FastGithub
  2. 点击”Releases”标签查看最新版本
  3. 下载适用于您操作系统的版本
  4. 解压下载的压缩包到合适的位置
  5. 以管理员身份运行 fastgithub.exe
  6. 软件会自动配置系统代理，使 GitHub 访问加速

5. 7-Zip 安装

7-Zip 是一款开源的文件压缩软件，支持多种压缩格式。

安装步骤：

  1. 访问 7-Zip 官网： https://www.7-zip.org/
  2. 根据您的系统选择下载版本（32位或64位）
  3. 下载完成后双击安装文件
  4. 按照安装向导提示完成安装
  5. 安装完成后可以在右键菜单中使用 7-Zip 功能

6. OpenUtau 安装

OpenUtau 是一款开源的歌声合成软件，类似于 Vocaloid。

安装步骤：

  1. 访问项目 GitHub 页面： https://github.com/stakira/OpenUtau
  2. 点击”Releases”标签查看最新版本
  3. 下载适用于您操作系统的版本（Windows 版本推荐下载 .exe 安装包）
  4. 下载完成后双击安装文件
  5. 按照安装向导完成安装过程
  6. 安装完成后需要下载音源库才能使用歌声合成功能

注意事项

  1. 从官方网站或可信来源下载软件，避免下载到恶意软件
  2. 安装前建议关闭杀毒软件，避免误报
  3. 安装过程中注意取消不必要的附加软件安装
  4. 定期更新软件以获得最新功能和安全补丁
  5. 对于需要配置环境变量的软件（如 Java），配置完成后需要重启计算机才能生效

PCL2 启动器安装与使用教程

发表于 2026-03-21 更新于 2026-04-14 分类于 游戏

从零开始，带你玩上 Minecraft Java 版
作者：AAA鱗遊SCROAM | 博客：blog.world123.top

写在前面

你是不是也遇到过这些情况？

  • 下载了 PCL2，打开却提示“未找到 Java”
  • 杀毒软件突然弹窗说“这是病毒”
  • 启动器下载游戏时卡住不动
  • 进游戏后黑屏、闪退、提示“内存不足”

别慌，这篇教程就是为你准备的。
我会用最直白的话，把每一步都拆开来讲，保证你跟着做就能玩上。

第一步：下载 PCL2 启动器

1.1 去哪下？

官方发布页（唯一安全来源）

  • 龙猫的百科：https://afdian.net/p/8e3e3f3c2d3e11ed8f6f52540025c377（内有下载链接）
  • 官方网站：https://pcl.ruanmao.net/
  • 或者直接去 GitHub：https://github.com/Hex-Dragon/PCL2/releases

千万别去百度搜“PCL2 下载”，前几个结果大概率是带广告、带病毒、甚至要你付费的假网站。

1.2 下载哪个文件？

你会看到两种文件：

  • Plain Craft Launcher 2.exe —— 直接可运行，推荐下载这个
  • Plain Craft Launcher 2.zip —— 压缩包，需要解压才能用

如果你下载的是 .exe，双击就能打开。
如果你下载的是 .zip，先右键解压到某个文件夹（比如 D:\PCL2），再双击里面的 Plain Craft Launcher 2.exe。

💡 分不清压缩包和普通文件？看图标：如果图标是一个带拉链的文件夹，那就是压缩包，需要解压。

第二步：安装 Java（必须！）

Minecraft Java 版是用 Java 语言写的，所以你的电脑里必须安装 Java 才能运行。

2.1 下载 Java

打开官网：https://www.java.com/zh-CN/download/
点击红色的“免费 Java 下载”按钮，下载安装程序。

⚠️ 别去百度搜“Java 下载”，那里全是广告和捆绑软件。

2.2 安装 Java

双击下载的安装程序，一路点“下一步”或“安装”，直到完成。
不需要修改任何路径，就让它装在默认位置。

2.3 验证是否安装成功

按下键盘上的 Win + R，输入 cmd，回车。
在黑色窗口里输入 java -version，然后回车。
如果看到类似 java version "1.8.0_xxx" 的字样，说明 Java 装好了。

如果提示“不是内部或外部命令”，说明 Java 没装好，或者安装时没选默认路径。这时候重新装一遍，什么都别改。

第三步：启动器设置与游戏下载

3.1 打开 PCL2，设置 Java

双击 PCL2 图标，第一次打开可能会弹出“未找到 Java”的提示。
别慌，点“设置” → 找到“Java 路径”那一栏，点击右边的 “自动搜索”。
启动器会自动找到你刚才安装的 Java，然后显示“已找到 Java”。

❗ 千万不要手动去桌面找一个叫 java.exe 的文件——Java 不是放在桌面用的！

3.2 下载游戏版本

回到主页，点击“下载” → “游戏版本”。
选择你想要玩的版本（比如 1.20.1），然后点击“安装”。
启动器会自动下载游戏文件，进度条走完就行。

🌐 如果下载速度很慢或一直失败，可能是网络问题。可以试试开个加速器（比如 Watt Toolkit），或者换个时间再下。

3.3 设置内存

点击“设置” → “游戏设置” → 找到“内存”选项。

  • 如果你的电脑内存是 8GB，建议分 4GB
  • 如果是 16GB，分 6-8GB
  • 分少了游戏会卡，分多了电脑本身会卡

简单记：**给 Minecraft 留一半左右的内存，但不要超过电脑总内存的 80%**。

第四步：启动游戏

点击主页的“启动”按钮。
第一次启动可能会慢一些，耐心等一会儿。

如果弹出杀毒软件警告（比如 Windows Defender 说“此应用可能有害”），点“更多信息” → “仍要运行”。
PCL2 被误报是家常便饭，因为它的行为和某些病毒相似（比如修改文件、联网下载），但只要你从官网下载的，就是安全的。

游戏窗口出来后，看到 Minecraft 主界面，就成功了！

常见问题（都是你可能会遇到的）

❓ 启动器提示“未找到 Java”

  • 回到第三步，点“自动搜索”。如果还是找不到，说明 Java 没装好，重新装一遍。

❓ 杀毒软件报毒，文件被删了

  • 打开 Windows 安全中心 → 病毒和威胁防护 → 管理设置 → 添加排除项 → 把 PCL2 的整个文件夹添加进去，然后重新下载 PCL2 放进去。

❓ 下载游戏时卡住不动

  • 检查网络，可以试试用 Watt Toolkit 加速（教程：GitHub访问优化）。

❓ 进游戏后黑屏或闪退

  • 可能是内存不够，去设置里调高一点。
  • 也可能是显卡驱动问题，更新显卡驱动试试（去显卡官网下载，别用驱动精灵）。

❓ 模组怎么装？

  • 下载 .jar 文件，放进 .minecraft/mods 文件夹（注意不要解压！）。
  • 如果还不会，等我的下一篇教程（已经在写了）。

写在最后

这篇教程是我七年 Minecraft 经验的浓缩。
如果你按步骤走还是有问题，别急着发帖问，先看看启动器的报错提示——它往往会告诉你答案。

实在不行，带上 logs/latest.log 文件来找我（发到博客评论区或群里），我会帮你“算一卦”。

记住：会玩之前，先学会自己修路。
祝你在方块世界里玩得开心！

电脑入门 08：怎么问问题别人才愿意帮你

发表于 2026-03-21 更新于 2026-04-10 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

问问题前，先做三件事

1. 读弹窗
弹窗上的字不是天书。它往往直接告诉了你问题。

弹窗内容 啥意思
内存不足 开的程序太多，关几个
缺少xxx.dll 软件没装完整，重装
无法连接 检查网络，或开VPN
访问被拒绝 右键以管理员身份运行
未找到Java 没装Java，或没让启动器自动搜索

2. 搜一下
把弹窗上的关键词复制下来，去百度、B站搜。大概率已经有人遇到过。

3. 自己试一下
重启电脑（能解决一半问题）、重装软件（能解决另一半）、换个网络（WiFi不行就开手机热点）。

问的时候，带上这些

一个有用的提问长这样：

【问题】PCL2启动时报“未找到Java”
【环境】Win11，笔记本
【试过】装了Java，点了自动搜索还是找不到
【截图】[用 Win+Shift+S 截的]

千万别这样问：

  • “电脑坏了，怎么办”
  • “PCL2打不开，求大佬”
  • [发一张全是摩尔纹的手机拍屏幕照片]

日志是啥？怎么发？

很多软件报错时会生成日志文件，里面记录了详细信息。

PCL2的日志在哪：打开PCL2文件夹 → .minecraft → logs → latest.log
用记事本打开，复制内容发出来就行。

最后

  • 把问题说清楚
  • 发完整报错信息
  • 说谢谢
  • 先自己搜过

别人帮你，是情分，不是义务。

电脑入门 07：电脑卡了怎么办

发表于 2026-03-21 更新于 2026-04-10 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

开机慢

按 Ctrl + Shift + Esc 打开任务管理器 → 启动 → 把不需要开机启动的软件禁用（微信、QQ、迅雷这些）

桌面上别放太多文件，也会拖慢开机。

运行卡

任务管理器 → 性能标签 → 看是CPU、内存还是磁盘满了
谁满就关谁。

如果内存经常满，考虑加内存条（老电脑的常见问题）。

C盘满了

C盘是系统盘，空间不够会变慢、没法更新。

清理方法：

  1. 右键C盘 → 属性 → 磁盘清理 → 勾选“临时文件”“回收站”“下载文件”（确认没重要东西）→ 确定
  2. 把桌面和“下载”文件夹里的文件移到D盘
  3. 卸载不用的软件
  4. 微信/QQ的设置里，把文件保存位置改到D盘

啥时候该重装系统？

  • 用了好几年，越来越慢
  • 中了病毒杀不干净
  • 频繁蓝屏
  • 换了新硬盘

重装前记得备份重要文件。

电脑入门 06：网连不上怎么办

发表于 2026-03-21 更新于 2026-04-10 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

先开CMD

按 Win + R，输入 cmd，回车。
会出来一个黑窗口，下面几条指令都在这里敲。

几个有用的命令

指令 干嘛的
ping 114.114.114.114 测能不能上网。有回复说明通，超时说明断
ping 192.168.1.1 测能不能连上路由器。通说明你和路由器没问题
ping www.baidu.com 测DNS正不正常。能ping通IP但ping不通百度，说明DNS有问题
ipconfig /flushdns 刷新DNS缓存。网站打不开时清一下
tracert www.baidu.com 跟踪路由，看数据包走到哪断了

举个例子

如果你 ping 114.114.114.114 超时，但 ping 192.168.1.1 能通，说明你家到运营商之间有问题——可能是欠费了，或者光猫红灯了。

VPN是啥？

VPN可以帮你访问一些境外网站（GitHub、Google、正版验证）。

注意：

  • 开着VPN时，上国内网站可能会变慢
  • 玩游戏时可能会卡
  • 用完记得关

如果只是想加速GitHub、Minecraft正版验证，用 Watt Toolkit（免费，不用翻墙）。

电脑入门 05：杀毒软件怎么选

发表于 2026-03-21 更新于 2026-04-13 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

Windows自带的够用吗？

够用。Windows自带的 Microsoft Defender 不卡电脑、不弹广告，对大部分人来说完全够了。

但如果你觉得它太敏感（比如动不动把PCL2删了），可以换个。

推荐两个

火绒

  • 安静，不弹窗
  • 有“弹窗拦截”功能，能屏蔽各种广告
  • 适合不想被烦的人

360国际版

  • 查杀能力强
  • 没广告（和国内版是两码事）
  • 适合追求安全的人

别用国内版的360安全卫士和腾讯电脑管家——广告多、全家桶、删不干净。

装了杀毒软件要注意啥？

  1. 不要同时装两个，会打架
  2. 报毒时先看来源。从官网下的PCL2、HMCL这些，通常是误报，点“添加信任”就行
  3. 不确定的：去 www.virustotal.com 查一下，几十个杀毒引擎一起扫

浏览器被劫持了咋办？

打开浏览器就弹广告？

方法一：右键桌面浏览器快捷方式 → 属性 → 看“目标”后面有没有多余的网址 → 有就删掉

方法二：浏览器设置里改主页

方法三：用火绒的“系统修复”扫一遍

电脑入门 04：软件怎么装怎么删

发表于 2026-03-21 更新于 2026-04-10 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

软件去哪下？

安全渠道：

  1. 官网（搜“软件名 + 官网”）
  2. Microsoft Store（Win10/11自带的商店）
  3. GitHub、SourceForge（技术类软件）

千万别去：

  • 百度前几个带“广告”字样的链接
  • 写着“高速下载”“极速下载”的按钮
  • 任何让你先下“下载器”的网站

那些“下载站”会给你下个几MB的“下载器”，运行后给你装一堆垃圾软件。看见文件名带“@”“_”的，直接删。

怎么安装？

  1. 双击安装包（.exe文件）
  2. 弹窗问“是否允许”，点**“是”**
  3. 一路“下一步”
  4. 关键一步：看到“自定义安装”或“高级安装”时，点进去，把里面“安装XX浏览器”“设为首页”的勾全部去掉
  5. 安装路径不用改（默认在C盘），除非你C盘快满了，才改到D盘
  6. 完成

桌面上的图标是啥？

安装完后桌面上出现的图标叫快捷方式（左下角有个小箭头）。

它只是一张名片，指向真正安装的软件。删掉快捷方式，软件还在。删掉软件，快捷方式就没用了。

怎么卸载？

不要直接把桌面图标拖进回收站！

正确方法：

  1. 按 Win + X → 应用和功能
  2. 找到要删的软件
  3. 点“卸载”

电脑入门 03：下载的压缩包怎么办

发表于 2026-03-21 更新于 2026-04-14 分类于 电脑入门

作者：AAA鱗遊SCROAM | 博客：blog.world123.top

什么是压缩包？

你下载的文件如果图标是个带拉链的文件夹，或者名字结尾是 .zip、.rar、.7z，那就是压缩包。

压缩包的作用：

  • 把一堆文件打包成一个，方便传给别人
  • 把文件变小，下载更快

怎么解压？

先装个解压软件。推荐：

  • WinRAR：经典好用，右键菜单一目了然。免费版会弹广告窗，点掉就行
  • 7-Zip：免费、没广告
  • 别用“好压”“360压缩”，有广告还绑垃圾

装好后，右键点压缩包 → 选 “解压到 [文件夹名]” → 等进度条走完，打开文件夹就能用了。

特别提醒：Minecraft 模组不要解压！

模组文件是 .jar 格式，虽然看起来像压缩包，但直接放进 mods 文件夹就行，千万别解压！
解压成一堆 class 文件反而会报错。

本大师见过太多人问“为什么模组装不上”了——就是手贱解压了。

文件和文件夹

文件就是一个个具体的东西：照片、文档、安装包。
文件夹就是装文件的“抽屉”。

一定要把后缀名打开

Windows默认隐藏后缀名，这是个大坑。

打开方法：随便开个文件夹 → 点“查看” → 勾选“文件扩展名”

这样你就能看到每个文件的真实身份了。
很多病毒会伪装成“照片.jpg.exe”，如果你只能看到“照片.jpg”，一点就中招。

文件放哪了？看路径

比如你存了一张照片在“D盘 → 我的文件 → 照片”里，它的位置就是：

这个叫路径，就是文件的“家庭住址”。

复制/剪切/粘贴

  • Ctrl + C 复制（原地留一份）
  • Ctrl + X 剪切（搬走）
  • Ctrl + V 粘贴（放到当前位置）

移动文件：剪切 → 到目标位置 → 粘贴
复制文件：复制 → 到目标位置 → 粘贴

桌面不是用来存文件的！

桌面放太多文件会：

  • 开机变慢
  • 重装系统时全丢

原则：桌面上只放快捷方式（左下角有小箭头那种），真正的文件都放到D盘或文档里。

电脑长啥样？

其实就两样东西：硬件（你摸得到的）和软件（你看不到的）。

硬件就是主机、显示器、键盘鼠标。笔记本的话，这些全塞在一个壳子里。

软件就是Windows系统，还有浏览器、微信、游戏这些。

桌面是啥？

开机后你看到的那个屏幕，叫桌面。

  • 左下角那个Windows图标 = 开始菜单，所有程序都在里面
  • 底下那条长条 = 任务栏，你打开的程序会显示在那里
  • 右下角 = 时间、网络、音量

鼠标怎么用？

  • 左键单击：选中
  • 左键双击：打开
  • 右键单击：弹出菜单（这个最有用，不知道点哪就点右键）
  • 滚轮：上下滚动

几个最常用的键盘快捷键

快捷键 干嘛的
Ctrl + C 复制
Ctrl + X 剪切
Ctrl + V 粘贴
Ctrl + Z 撤销（后悔药）
Ctrl + A 全选
Win + E 打开文件管理器
Win + Shift + S 截图（别再用手机拍了！）

截图一定要学会

按 Win + Shift + S，屏幕变暗，拉个框，截图就自动存好了。
在聊天窗口按 Ctrl + V 就能发出去。

别再拿手机拍屏幕了，拍出来全是摩尔纹，谁也看不清。
